## このドキュメントの使い方

本ドキュメントは、ブラウザで各Webサイトにアクセスして、必要なマクロ分析データを手動で記録するためのチェックリストです。

各項目について、以下の手順で実施してください：

1. 提示されたURLをブラウザで開く
2. 該当する値を記録
3. 本ドキュメント内の【記録欄】に値を記入
4. すべての値を入力後、`data/macro_data_template.json` に転記

## 【データ転記作業】

すべての【記録欄】に値を入力したら、以下のファイルに転記してください：

```bash
# JSONファイルの編集
vi data/macro_data_template.json

# または Python / テキストエディタで以下を編集：
cp data/macro_data_template.json reports/2026-06-24_macro_data.json
```

転記完了後：

```bash
# ファイル検証
python3 -m json.tool reports/2026-06-24_macro_data.json

# Git へのコミット
git add reports/2026-06-24_macro_data.json
git commit -m "Add macro analysis data for 2026-06-24"
git push origin main
```
