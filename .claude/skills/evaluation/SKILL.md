# 銘柄評価

スクリーニングで選出された銘柄を、テンバガー候補としての独自軸で多角的に評価します。（最大10銘柄）

## 評価軸

`reports/YYYY-MM-DD_screening.md`の最新日付でスクリーニングされた各銘柄を以下の4軸で評価し、それぞれ5段階（★1〜★5）で採点する。
また、下記評価軸はAeroEdge：7409の成長に倣った評価軸。
※AeroEdge：7409の構造的成長ストーリーの定義・参考事例は `.claude/skills/evaluation/references/aeroedge_7409.md` を参照すること。

### 軸1：ビジネスモデルの根本的進化
- ビジネスモデルが根本的に進化しているか（単なる値上げ・コスト削減・一過性のニュースによる株価上昇でない）
- サプライチェーン上の立場がTier2からTier1へ
- 独自技術の確立（自社技術により、材料供給可能に）

### 軸2：独自技術
- 他社には真似できない独自技術を持っているか
- 他社依存リスクはないか（特定サプライヤーへの依存、ライセンス依存など）
- 特許・独自技術による参入障壁はあるか

### 軸3：市場の成長性と安定供給
- 属する市場が構造的成長段階にあるか
- 市場内でのポジション（シェア・ニッチ支配）は安定しているか
- 需要の継続性・可視性はあるか

### 軸4：量産案件・受注の確度
- 新規の量産案件を獲得済みか、または受注見込みが高いか
- パイプラインの厚みと顧客の質
- 売上の再現性・予測可能性

### 軸5：収益性と希少性
- 営業利益率の水準と改善トレンド
- 同等の事業モデルを持つ競合が少ないか
- バリュエーション（PER・PBR）は割安か、成長に見合っているか

## 総合判定

| 判定 | 基準 |
|------|------|
| S：即買い候補 | 全軸★4以上、構造的成長ストーリーが明確 |
| A：詳細調査推奨 | 平均★3.5以上、1軸以上★5 |
| B：ウォッチ継続 | 平均★3以上、要素はあるが確度不足 |
| C：見送り | 平均★3未満、または致命的懸念あり |
