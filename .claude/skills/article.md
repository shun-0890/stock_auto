# 記事執筆

スクリーニング・評価・詳細調査の結果を統合し、読みやすい投資分析記事を作成します。

## 前提

- 当日の `reports/YYYY-MM-DD_screening.md`、`reports/YYYY-MM-DD_evaluation.md`、および必要に応じて `reports/YYYY-MM-DD_deepdive_XXXX.md` を参照する
- 対象読者：自分自身の投資判断記録、および同様の投資方針を持つ投資家

## 記事構成

### 1. 当日マクロサマリー（300字以内）
- 市場全体の動きと注目テーマを簡潔に
- リスクオン/オフの判定と根拠

### 2. 本日の注目銘柄（スクリーニング結果）
- 選出銘柄を一覧で紹介
- 各銘柄に1〜2行のキャッチコピー的なコメント

### 3. 銘柄評価ハイライト
- S・A判定銘柄を重点的に紹介
- 評価軸ごとの強み・弱みを端的に記述
- C判定銘柄は「見送り理由」を一言添える

### 4. 詳細調査銘柄の深掘りレポート（詳細調査実施分のみ）
- 詳細調査レポートの内容をより読みやすく再構成
- 投資判断と根拠を明確に記述

### 5. 本日のアクションプラン
- 具体的に何をすべきか（買い・ウォッチ・見送り）
- ポジション構成の変更提案があれば記載

## 執筆スタイル

- 忖度なし・客観的・根拠ベース
- 専門用語は使うが、根拠を必ず添える
- 「〜と思われる」などの曖昧表現を避け、断言する
- 箇条書きと表を積極的に使い、視認性を高める

## 出力形式

`reports/YYYY-MM-DD_article.md` に保存する。

```
# 投資分析レポート：YYYY-MM-DD

## 本日のマクロ環境
...

## 本日の注目銘柄
...

## 銘柄評価ハイライト
...

## 詳細調査：XXXX 〇〇〇〇
...

## 本日のアクションプラン
...
```
