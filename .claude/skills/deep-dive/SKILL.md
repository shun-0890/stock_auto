# 詳細調査

評価レポートで「S：即買い候補」または「A：詳細調査推奨」となった銘柄を深掘り調査します。
調査分析は忖度なしに実施し、詳細調査した結果微妙な結果であってもあるがままの情報を提示すること。

## 調査対象の指定

・自動実行
evaluation.mdのルールに則って評価した銘柄のうち、上位2銘柄を調査対象として詳細調査を実行する。

・手動実行
ユーザーから銘柄コードまたは銘柄名を受け取り、1銘柄ずつ詳細調査を実施する。

## 調査項目

※1 - 6の評価を、一言の総評に加えて定量評価も（定量評価については以下）
※各セクションは箇条書き5項目以内に収めること

| 判定 |
|------|
| S+：欠点なしの最高評価 |
| S：同業界においても比類なき高評価 |
| A：高水準。普通に高評価 |
| B：短所あり |
| C：見逃してはならない難点あり |

### 1. 構造的成長ストーリー（evaluation引継ぎ＋深掘り）

**新たに調査は行わず、`reports/YYYY-MM-DD_evaluation.md` の当該銘柄の評価結果（5軸のスコアとコメント）をそのまま引用・要約する。**
その上で、2〜6の新規調査結果と照らし合わせて評価の妥当性・更新点があれば追記する。

引用対象：
- ビジネスモデルの根本的進化（★）
- 独自技術（★）
- 市場成長性・安定供給（★）
- 量産案件・受注確度（★）
- 収益性・希少性（★）

### 2. 事業概要と競合優位性
- 主力製品・サービスの詳細
- 競合他社との差別化ポイント
- 顧客・取引先の質（大手か、分散しているか）
- マクロとの絡み（日本政府の国策との絡み、市場トレンドとの絡み）

### 3. 財務分析（直近3期以上）
- 売上高・営業利益・純利益の推移
- 営業利益率・ROEの推移
- フリーキャッシュフロー
- 有利子負債・自己資本比率
- 「伸びる銘柄のブレイク」か「テーマで持ち上げられただけ」かの見立て

### 4. 成長ドライバーの検証
- 直近の受注・量産案件の具体的内容
- IR資料・決算説明資料から読み取れる経営陣のコミットメント
- 中期経営計画の目標と達成可能性
- 直近のニュース（株価への影響のあるニュース）

### 5. リスク分析
- 最大のダウンサイドシナリオ
- 競合の参入・代替技術リスク
- 顧客集中リスク
- マクロリスク（為替・金利・規制）

### 6. バリュエーション＆テクニカル
- 現在のPER・PBR・EV/EBITDA、同業他社比較
- 成長を織り込んだ適正株価の試算（保守・中立・強気シナリオ）
- 現在の株価位置（52週レンジ内）・出来高トレンド・移動平均線との位置関係
- 推奨エントリーゾーン

## 投資判断

1〜6の総合から導く。以下を明記する：
- **買い推奨か否か**（Yes/No）
- **エントリータイミング**（どのタイミングでエントリーすべきか。また買い増しのタイミングも。）
- **推奨ポジションサイズ**（ポートフォリオの何%か）
- **目標株価**（12〜24ヶ月）
- **損切りライン**
- **モニタリングすべきトリガー**（上方修正、量産開始発表など。時系列順にわかりやすく出力。）
※エントリータイミングについては`reports/YYYY-MM-DD_macro.md`の最新日付のレポートも参照して決める
