# 詳細調査

評価レポートで「S：即買い候補」または「A：詳細調査推奨」となった銘柄を深掘り調査します。
調査分析は忖度なしに実施し、詳細調査した結果微妙な結果であってもあるがままの情報を提示すること。

## 調査対象の指定

・自動実行
evaluation.mdのルールに則って評価した銘柄のうち、★4以上の上位2銘柄を調査対象として詳細調査を実行する。

・手動実行
ユーザーから銘柄コードまたは銘柄名を受け取り、1銘柄ずつ詳細調査を実施する。

## 調査項目

※1 - 7の評価を、一言の総評に加えて定量評価も（定量評価については以下）

| 判定 |
|------|
| S+：欠点なしの最高評価 |
| S：同業界においても比類なき高評価 |
| A：高水準。普通に高評価 |
| B：短所あり |
| C：見逃してはならない難点あり |

### 1. 構造的成長ストーリー判定

※構造的成長ストーリーの定義・参考事例は `.claude/skills/references/aeroedge_7409.md` を参照すること。

- 独自の技術あるか（他社依存による供給ストップ等ないか）
→ビジネスモデルの根本的な進化見えるか
- 成長段階にある市場なのか。その市場での安定供給が見込めるか。
- 新規の量産案件ありそうか
- 利益率成長と希少性あるか

### 2. 事業概要と競合優位性
- 主力製品・サービスの詳細
- 競合他社との差別化ポイント
- 顧客・取引先の質（大手か、分散しているか）
- マクロとの絡み（日本政府の国策との絡み、市場トレンドとの絡み）

### 3. 財務分析（直近3期以上）
- 売上高・営業利益・純利益の推移
- 営業利益率・ROEの推移
- フリーキャッシュフロー
- 有利子負債・自己資本比率
- 配当方針
- 「伸びる銘柄のブレイク」か「テーマで持ち上げられただけ」かの見立て

### 4. 成長ドライバーの検証
- 直近の受注・量産案件の具体的内容
- IR資料・決算説明資料から読み取れる経営陣のコミットメント
- 中期経営計画の目標と達成可能性
- 直近のニュース（株価への影響のあるニュース）

### 5. リスク分析
- 最大のダウンサイドシナリオ
- 競合の参入・代替技術リスク
- 顧客集中リスク
- マクロリスク（為替・金利・規制）

### 6. バリュエーション
- 現在のPER・PBR・EV/EBITDA
- 同業他社比較
- 成長を織り込んだ適正株価の試算（保守・中立・強気シナリオ）

### 7. テクニカル確認
- 現在の株価位置（52週レンジ内）
- 出来高トレンド
- 移動平均線との位置関係
- 推奨エントリーゾーン

## 投資判断

以下を明記する：
- **買い推奨か否か**（Yes/No）
- **エントリータイミング**（どのタイミングでエントリーすべきか。また買い増しのタイミングも。）
- **推奨ポジションサイズ**（ポートフォリオの何%か）
- **目標株価**（12〜24ヶ月）
- **損切りライン**
- **モニタリングすべきトリガー**（上方修正、量産開始発表など。時系列順にわかりやすく出力。）
