# 保有銘柄レビュー

現在保有している銘柄ごとに情報収集・分析を実施し、「損切」「利益確定」「押し目買い」「ホールド継続」のいずれかを推奨する。
忖度なし・感情なしで、あるがままの状況判断を提示すること。

## 実行トリガー

ユーザーから保有銘柄リストを受け取るか、`mylist/holdings.csv` が存在する場合はそれを読み込む。

### holdings.csv フォーマット（例）

```csv
銘柄コード,銘柄名,取得単価,保有株数
7409,AeroEdge,3200,100
XXXX,〇〇〇〇,YYYY,ZZZ
```

保有リストが提供されない場合はユーザーに確認する。

## 情報収集項目（銘柄ごと）

### 1. 現在の株価・テクニカル状況
- 現在株価・前日比・出来高
- 52週高値・安値と現在株価の位置
- 主要移動平均線（25日・75日・200日）との乖離
- 直近の出来高変化（増加 / 減少 / 急増）

### 2. ファンダメンタルの変化確認
- 直近の決算・業績修正の有無
- 構造的成長ストーリーは当初評価から変化していないか
- IR・適時開示の最新情報

### 3. 直近ニュース・カタリスト確認
- 株価に影響しうる直近1ヶ月のニュース
- 競合他社の動向
- セクター全体のトレンド変化

### 4. マクロ環境との整合性
- 現在のマクロ環境（金利・為替・リスクオン/オフ）が当該銘柄に追い風か逆風か
- セクターローテーションの方向性

## 推奨アクション判定基準

### 損切
以下のいずれかを満たす場合：
- 構造的成長ストーリーが崩壊した（技術優位性の喪失、主要顧客喪失、競合参入）
- 業績が構造的に悪化している（一時的でなくビジネスモデルの毀損）
- 明らかな悪材料が出た
- テクニカル的に主要サポートを明確に割り込み、反発の兆候なし（25日平均線を5日平均線が下回る（デッドクロス）等）
- ファンダメンタルの悪化と同時にマクロ情勢の悪化

### 利益確定（全部 or 一部）
以下のいずれかを満たす場合：
- 詳細調査レポートで設定した目標株価に到達
- 天井圏で大陰線が出現
  ※天井圏の定義（以下のいずれか）
  　・直近4週間の新高値ゾーン
  　・短期/長期RSIが共に70%以上
- バリュエーションが成長を完全に織り込んでいる（割高水準）
- より優先度の高い投資先へのリバランスが必要
- 三空後に出来高急増を伴った陰線出現

### 押し目買い（買い増し）
- 構造的成長ストーリーに変化なし
- 株価下落がマクロ要因や需給によるもので、ファンダメンタルに影響なし
上記すべて、かつ以下いずれか一つの条件を満たす場合
- テクニカル的なサポートゾーン付近（25日平均線を5日平均線が超える（ゴールデンクロス）等）
- マクロ情勢の回復
- 底値圏にて大陽線出現
　※底値圏の定義（以下のいずれか）
　　・直近4週間で-20％以上の株価下落
　　・短期/長期RSIが共に30%未満

### ホールド継続
- 上記いずれにも該当しない
- 次のカタリストを待つフェーズ
