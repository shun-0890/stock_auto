# 再エントリー検討

過去に手放した銘柄について、再エントリーの可否を検討します。
「一度手放した」という事実に引きずられず、現時点での客観的な評価を行うこと。

## 実行トリガー

`mylist/old.csv` を読み込み、売却済み銘柄リストを取得する。
ファイルが存在しない・空の場合は処理を終了する。

### old.csv フォーマット（例）

```csv
銘柄コード,銘柄名,売却日時,売却理由
7409,AeroEdge,2026-03-10,利益確定
XXXX,〇〇〇〇,YYYY-MM-DD,損切 / 利益確定 / リバランス
```

※ `売却理由` 列は任意。記載がない場合は「不明」として扱う。

## 分析項目（銘柄ごと）

### 1. 売却時からの状況変化
- 売却日時点と現在の株価比較
- 売却後に発生した主要ニュース・決算・IR
- ストーリーの変化（好転 / 悪化 / 変化なし）

### 2. 構造的成長ストーリーの現状確認
- evaluation.md の評価軸（5軸）で現時点を再評価
- 売却理由が「損切」の場合：崩壊したストーリーが修復されているか
- 売却理由が「利益確定」の場合：再び押し目圏に入っているか

### 3. 現在のテクニカル状況
- 現在株価・52週レンジ内の位置
- 主要移動平均線との乖離
- 出来高トレンド

### 4. 再エントリーのタイミング評価
- 現在の株価水準は仕込みやすいゾーンか
- 次のカタリスト（決算・量産開始・IR等）のタイミング
- マクロ環境との整合性

### 5. 現ポートフォリオとの整合性
- `mylist/holdings.csv` を参照し、現在の保有銘柄・集中度を確認
- 追加するとポートフォリオが何銘柄になるか（目安：最大5銘柄）
- セクター分散への影響

## 再エントリー判定基準

### 即時再エントリー推奨
以下をすべて満たす場合：
- 構造的成長ストーリーが健在（または売却時より強化されている）
- テクニカル的に良い仕込みゾーン（底値圏で好材料を伴った上昇、25日平均線を5日平均線が上回る（ゴールデンクロス））
- 次のカタリストまでの時間軸が明確

### 監視継続（ウォッチリスト入り）
以下のいずれかに該当する場合：
- ストーリーは健在だが株価が割高水準
- カタリスト待ち（次の決算・量産開始等まで様子見）
- ポートフォリオの空きができたら優先度高

### 見送り
以下のいずれかに該当する場合：
- 損切した理由（ストーリー崩壊）が解消されていない
- より優先度の高い銘柄が現ポートフォリオに存在する
- 構造的成長ストーリーの優位性が失われた
