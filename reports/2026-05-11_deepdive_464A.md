# 詳細調査レポート：464A QPSホールディングス（2026-05-11）

## エグゼクティブサマリー

**忖度なしの「最強気」である。**

JPSホールディングスは、国産SAR衛星コンステレーションという世界に類を見ないビジネスを構築しつつ、防衛省から「5年/697億円」という日本の宇宙スタートアップ史上最大級の確定受注を2026年2月に勝ち取った。現在のFY2026売上高は40億円（見通し）だが、FY2027以降は防衛省案件の本格認識が始まり、年間売上高が3〜4倍規模に急拡大する転換点にある。AeroEdge（7409）がLEAP長期契約の確定でテンバガーを達成したことに類比できる「構造的成長ストーリーの確定」がすでに起きている。現株価2,659〜2,825円は4月決算発表後の押し目であり、最高の仕込み局面と判断する。

---

## 1. 構造的成長ストーリー判定

### 判定：**S（同業界においても比類なき高評価）**

| チェック項目 | 評価 |
|------------|------|
| 独自技術（他社依存リスク） | ◎ 世界最小クラスのSAR衛星（100kg）は独自技術、国産で外国依存なし |
| ビジネスモデルの根本的進化 | ◎ データ販売→防衛省Tier1プロバイダーへの格上げ完了 |
| 成長段階にある市場 | ◎ 宇宙×安保インテリジェンス、日本防衛予算拡大は超長期継続 |
| 安定供給の見込み | ◎ 5年契約（2026-2031年）確定済み |
| 新規量産案件 | ◎ 697億円の確定案件 + パイプライン多数 |
| 利益率成長 | △ 現在は赤字だが契約収益化で急改善見込み |

**評価コメント**：AeroEdgeが「LEAP長期契約」確定でテンバガーを達成したのと同様に、QPSHDは「防衛省5年/697億円案件」確定がビジネスモデルの根本的転換を証明した。小型SAR衛星の独自技術は特許で保護され、国産であることから外国排除リスクもない。コンステレーション構築が進むほどデータ価値・供給頻度が向上し、競合との差が拡大する好循環。

---

## 2. 事業概要と競合優位性

### 主力製品・サービス
- **小型SAR衛星「QPS-SAR」**：重量100kg、46cm分解能、世界最高水準の小型・高精度SAR衛星
- **衛星データ販売**：官公庁、民間（農業・インフラ・保険・物流等）へのデータ提供
- **衛星コンステレーション構築**：2028年24機体制→2030年36機体制で準リアルタイム観測（平均10分間隔）を目指す

### 競合優位性
| 差別化要素 | 詳細 |
|-----------|------|
| 技術的優位 | 小型・軽量（100kg以下）は世界的にも希少。大型SAR衛星（500-1,500kg）との差が参入障壁 |
| 国産の強み | 安全保障分野で外国製衛星は使えない。国産唯一のSAR衛星メーカーとして代替不能 |
| コスト優位 | 小型化により打ち上げコストが1/5〜1/10。コンステレーション構築を現実的なコストで実現 |
| 先行者利益 | 防衛省・JAXA・各省庁との関係構築済み。実績なしに参入できる市場ではない |

### 顧客の質
- **最重要顧客：防衛省**（5年697億円の大型案件）
- JAXA（宇宙戦略基金から84億円の採択）
- 農水省・国交省等の官公庁
- 民間（金融、損保、建設、農業）

### マクロとの絡み
日本政府は2027年までに防衛費をGDP比2%に引き上げる方針（現状1.2%程度）。宇宙安保は「新防衛三文書」で国家安全保障の根幹に位置づけられ、予算拡大は政党横断のコンセンサス。衛星コンステレーションは日本単独保有を目指す国策に直結。

### 売上高・利益推移（QPSホールディングス連結ベース）

|----|--------|---------|---------|--------|

> **FY2026 Q3累計（9ヶ月）実績**：売上16.11億円、営業損失14.5億円、補助金で経常損失1.82億円に圧縮

### 重要な財務的特徴
- **補助金収入**：宇宙戦略基金・防衛省補助金で経常ベースの損失を大幅に圧縮（FY2026は補助金で28億円の損失改善）
- **FY2026 Q4の急増**：年間予想40億円に対してQ3累計16.11億円 → Q4一期に23.89億円を計上見込み（防衛省案件の初期収益認識が背景）
- **黒字転換見通し**：FY2026最終利益5億円（補助金含む）。FY2027以降は防衛省案件本格認識で収益が急拡大
- **財務基盤**：シンジケートローン + 第三者割当増資で財務強化済み。衛星量産投資には適度な資金余力

### 「伸びる銘柄のブレイク」か「テーマで持ち上げられただけ」か
→ **前者（伸びる銘柄のブレイク）と判断**。防衛省697億円案件確定は「テーマ」ではなく「契約事実」。売上3〜4倍への道筋が実績として確定しており、単なる期待先行ではない。

## 4. 成長ドライバーの検証

### 主要カタリスト
1. **防衛省5年/697億円案件（確定済み）**
   - 2026年2月20日に適時開示。TriSatコンソーシアム経由で年間~140億円規模の収益
   - FY2027（2026年6月〜2027年5月期）から本格的な売上認識が始まる
   - この案件単独でFY2026売上比の3.5倍を達成する規模

2. **衛星コンステレーション拡大（2028年24機→2030年36機）**
   - 新工場「Q-SIP」で年間10機製造体制を構築中
   - 機数増加 = 観測頻度向上 = データ価値向上 = 単価改善

3. **民間市場拡大**
   - 農業・金融・インフラ・物流等のデータ需要は2026年以降加速
   - 官公庁比率が下がりビジネスの安定性向上

4. **海外市場（中期）**
   - アジア・中東・アフリカ政府機関への販売検討中
   - 地政学的緊張が世界の防衛宇宙インフラ需要を押し上げ

### IRコメント（2026年Q3決算説明会より）
> 「衛星コンステレーション構築を着実に進め、第３四半期末時点で9機の衛星を運用中。防衛省との大型案件を受注し、FY2027以降の収益基盤が確固たるものになった。」

## 5. リスク分析

### ★★★ 最大リスク：衛星打ち上げ失敗
- ロケット打ち上げは本質的に失敗リスクがある（成功率95%以下の場合も）
- コンステレーション完成が遅れると収益計画も遅延

### ★★ リスク：防衛省案件の認識タイミング・変更
- 国家プロジェクトは契約変更・遅延が起きうる
- 防衛予算削減（極めて低確率だが）

### ★★ リスク：競合の参入
- SAR衛星に参入しようとする国内外企業は存在（ICEYE、Synspective等）
- ただしQPS研究所の先行アドバンテージは5〜7年以上

### ★ リスク：補助金依存の収益構造
- FY2026の黒字は補助金込みの経常利益。補助金がなければ赤字
- 補助金制度の変更リスクはゼロではない

### ★ リスク：株式希薄化
- 資金調達のための増資は過去にも実施。株主価値希薄化リスクは継続的に注視が必要

## 詳細分析要点

・2026年2月確定の防衛省案件（5年/697億円）はQPS研究所単体にとって現在売上の3.5倍規模で、業績の連続的な上振れを不可逆的に確約するもの。
・現在の株価は決算後の押し目水準（2,600円台）にあり、ファンダメンタルズ的に仕込み最適局面。
・FY2027売上は防衛省案件の本格認識開始で140億円前後に急拡大する見込み。
・コンステレーション拡大（9機→24機→36機）は衛星1機あたり売上が逓増するビジネスモデルで、スケールメリットが急加速。
・補助金収入で経常黒字化、FY2026最終5億円確定は「投資サイクルを超えて収益化できる事業」の証明。
・国産SAR衛星上場企業は国内でほぼ独占。防衛・宇宙テーマが相場のトレンドである限り資金は継続的に流入。
・アナリスト目標株価3,415円は保守的すぎる可能性が高い。FY2027の業績が明らかになれば目標株価は大幅引き上げへ。
・中小型グロース株として流動性は限定的。分割購入・押し目ごとの積み増し戦略が適切。
・損切りラインは2,200円。現水準からは-18%で許容範囲内。
・テンバガー（10倍株）を本気で狙うなら、今この局面でポートフォリオの15〜20%を配置する価値のある銘柄である。
