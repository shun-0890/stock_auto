# 詳細調査レポート：153A カウリス（2026-05-20）

## エグゼクティブサマリー

**忖度なしの総評：「本気でテンバガーを狙うポートフォリオに入れるべき、希少かつ本物のディフェンシブSaaSである。」**

チャーン率0.28%・ARR+24.3%・Q1営業利益+33.9%という数字は、「成長が本物であり顧客が逃げない」ことを同時に証明している。業界横断の不正端末情報DBというネットワーク効果は、NTTデータですら参入から5年では追いつけない構造的優位性だ。加えて、Grid Data KYC（全国10社送配電事業者との独占データ連携）という第2の成長軸が具体化し始めており、現在の時価総額99億円は過小評価の領域にある。本日の金利ショック（米30年5.19%・グロース250 −3.4%）の逆風下でも規制ドリブンの需要は揺るがない。PER60倍という高さが唯一の弱点だが、24%成長継続を前提とすると2〜3年で正当化される水準だ。

## 2. 事業概要と競合優位性

- **Fraud Alert（主力・ストック型収益9割超）**：月間550億件超のレコードを監視。250以上のパラメータで口座開設・ログイン・入出金の3フェーズをリアルタイム監視。国内金融機関から不正端末情報を横断収集・共有するDB構造により、利用機関が増えるほど精度が指数的に向上するネットワーク効果を保有
- **Grid Data KYC（第2の柱・2025年9月開始）**：全国10社の一般送配電事業者（事実上の国家インフラ）が保有する約8,600万世帯分の電力契約情報を金融機関の顧客情報と照合。UI銀行での実証実験で凍結口座の「空き家8.4%・名義人不一致26.0%」を検出——競合が同等データを集めることは構造的に不可能
- **競合との差別化核心**：NTTデータ（CAFIS Brain）等の大手SIerは「金融機関ごとのオーダーメイドシステム」であり、カウリスの「業界横断DB＋クラウドSaaS」モデルとは本質的に異なる。大手が参入しても横断DBの構築には数年と金融機関全社の同意が必要
- **顧客ロック**：チャーン率0.28%は世界的SaaSトップクラス水準（優良SaaSの基準は1%以下）。年間1,000万件超の不正取引防止実績が顧客の解約を事実上不可能にしている
- **ASEAN展開**：シンガポール合弁会社設立予定（2026年内）。東南アジアの地場銀行とPoC進行中。次の成長軸として具体化段階

## 4. 成長ドライバーの検証

- **既存顧客へのアップセル加速**：2026年Q1は証券業界での不正アクセス多発を受けた既存顧客への追加契約が主な成長ドライバー。需要が外部環境悪化（犯罪増加）でむしろ増加するという逆張り的ディフェンシブ特性を実証
- **Grid Data KYCの展開加速**：UI銀行での実証実験成果（凍結口座の効率的検出）が定量的に示され、金融機関への横展開フェーズへ。全国10送配電会社との独占的データ連携は2〜3年で大きな売上柱に育つポテンシャル
- **FATF第5次対日審査対応（2024年〜2026年）**：日本政府がFATFの勧告に対応するため金融機関へのAML・KYC強化を義務付け中。この規制強化の最大受益者がカウリスであり、「やらざるを得ない需要」が急増している
- **ASEAN展開（シンガポール合弁・2026年内）**：東南アジアの金融犯罪対策需要は日本の5〜10年遅れで急速に高まっており、最初のPoC成功から受注までの時間軸が短い
- **経営陣のコミットメント**：島津CEO自身がASEAN展開・Grid Data KYCの戦略的重要性を繰り返し発信。中期でARRを2〜3倍にするロードマップが具体的に語られている

## 5. リスク分析

- **PER60倍のバリュエーションリスク**：本日の金利ショック（米30年5.19%・グロース250 −3.4%）のような局面では、高PER小型グロース株は市場全体の波に飲み込まれて一時的に急落する。成長が鈍化した瞬間の株価下落幅が大きい
- **大手IT系競合の本格参入**：NTTデータが横断型DBを構築しにくる場合、5〜7年のタイムラグはあるが将来の競争圧力となりうる。ただし業界横断DBは「最初に作った者が勝つ」ネットワーク効果産業であり、時間的優位は絶大
- **小型株の流動性リスク**：時価総額99億円では機関投資家が大口でポジションを組みにくく、売買時のスプレッドリスクがある。急落局面での逃げ足が遅くなる可能性
- **ASEAN展開の不確実性**：現地の規制環境・ローカルプレイヤーとの競合・文化的差異がPoC成功→本格受注の転換を遅らせる可能性がある。海外展開コストが先行しすぎると利益率が一時的に毀損
- **FY2025/12期の「成長鈍化」誤解リスク**：会社予想の営業利益が前期比▲37.5%減と保守的に設定されているため、数字だけ見ると減速に見える。実態は先行投資の費用計上であり、Q1+33.9%が示す通り成長は加速しているが、投資家の誤解が株価下押し要因になる局面があり得る
