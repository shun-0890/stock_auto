# 詳細調査レポート：3698 CRI・ミドルウェア（2026-06-26）

## エグゼクティブサマリー

**忖度なし評価：「稀有な参入障壁×SDV市場の大波。時価総額50〜60億円は著しく過小評価されている可能性がある。」**

ゲーム音響・映像ミドルウェア「CRIWARE」で業界標準を確立した企業が、自動車SDV（ソフトウェア定義型自動車）市場に静かに進出している。車載向け搭載台数が前年同期比3倍（321万台/H1）に急拡大し、2期連続最高益更新中。競合不在の独占的ポジションと小型時価総額の組み合わせが、テンバガー候補としての最大の魅力。
量産案件の具体的な可視性と、H1での投資先行による利益一時減少がリスクではあるが、H2回復が見込まれており、長期目線でのポジション構築を推奨する。

## 2. 事業概要と競合優位性

- **CRIWARE**: 音声・映像・リップシンク・触覚フィードバック対応のミドルウェア群。ドラゴンクエスト、ファイナルファンタジー、バイオハザードなど日本の主要IPほぼ全てで採用。業界のデファクトスタンダード。
- **展開領域の拡大**: ゲーム（コア）→ 遊技機（パチンコ等）→ カラオケ → **車載（SDV）** → 家電・IoT → 監視カメラ → Webビデオ → 医療機器
- **競合優位性**: ゲームエンジン（Unity/Unreal Engine）への深いネイティブ統合。30年超の実績と信頼性。一度採用されたゲームタイトルは次作でも継続採用（ブランドロイヤルティ）。
- **車載での実績**: CRIWARE for Mobility。2024年度上期の搭載車出荷台数は前年同期比約3倍（321万台）を達成し、累計600万台突破（2024年3月末）。

## 4. 成長ドライバーの検証

- **車載SDV市場の爆発的拡大**: SDV（ソフトウェア定義型自動車）市場は2030年に世界2,000億ドル規模予測。CRIWAREはすでに主要自動車メーカー向けに展開済み。搭載台数の3倍成長は実需を反映
- **ゲーム海外展開**: 海外ゲーム市場（特に米国・欧州）へのCRIWARE拡販を実行中。H1で前倒し投資→H2回復の構造
- **Q1 FY2026**: 「営業利益が前年同期比3.7倍に拡大」という報告あり（gamebiz）。H1全体での減少との矛盾が気になるが、Q1に強い業績があった可能性
- **市場変更（グロース→スタンダード）**: 2026年2月に達成。機関投資家の購入ハードル低下、株主層拡大に寄与
- **経営陣コミットメント**: 国際展開への前倒し投資を敢行しつつ通期増収増益予想を維持している点は自信の表れ

## 5. リスク分析

- **最大リスク: Unity/Unreal Engineへの統合深化**: 両エンジンが音声処理を内製化した場合、CRIWAREの優位性が低下する可能性。ただし、30年分の実績・チューニングの蓄積は簡単には代替できない
- **車載での欧米プレイヤー競合**: 欧米系の車載ソフトウェア企業（Qualcomm Snapdragon等）が音声処理領域に進出した場合
- **顧客集中リスク**: 主要ゲームパブリッシャー（任天堂、スクウェア・エニックス等）への依存度が高い可能性。1社の離脱で業績変化
- **規模リスク**: 従業員173名の小規模企業。大型案件獲得・実行には人的リソースが制約になりうる
- **マクロリスク**: 本日（6/26）の大幅な市場急落（-4.15%）でリスクオフが続く場合、小型グロース株全般に売りが波及

## 6. バリュエーション＆テクニカル

- **現在PER（予）**: 約13〜14倍（時価総額60〜65億円 / FY2026予想NI約4.6億円）
- **同業比較**: ゲーム関連ソフトウェア企業のPERは一般に20〜40倍。CRIの現状PER13〜14倍は著しく割安
- **適正株価試算**:
  - 保守シナリオ（成長鈍化）: FY2027 NI 5億円 × 15倍 = 75億円 → 株価1,465円（+20〜35%）
  - 中立シナリオ（現状継続）: FY2027 NI 5.5億円 × 22倍 = 121億円 → 株価2,363円（+96%）
  - 強気シナリオ（SDV爆発）: FY2028 NI 8〜10億円 × 30倍 = 240〜300億円 → 株価4,700〜5,860円（+4〜5倍）
- **テンバガー路線（10倍＝600億円市場）**: 車載CRIWARE が年間5,000万台超え＋IoT急拡大で売上100億円到達、NI 20億円× 30倍 = 600億円市場。5〜7年での実現は挑戦的だが不可能ではない
- **現在の株価位置**: 本日急落前の6/24時点で1,205円（52週高値圏）。本日市場急落で1,050〜1,150円程度に調整か
- **52週レンジ**: 1,000円以下（下半期）→ 1,205円（6月24日）。ブレイク初動の可能性
- **出来高**: 6/24に+7.4%の大幅上昇、出来高増加を伴ったと考えられる → 質の良いブレイク
- **推奨エントリーゾーン**: 1,050〜1,150円（本日急落での押し目）
