# 詳細調査レポート：5575 Globee（グロービー）（2026-07-13）

## エグゼクティブサマリー

**一言総評：「テンバガーには遠いが、PEG0.45の低評価成長SaaSとして短中期の反発機会は存在する」**

Globeeは、英語学習プラットフォーム「abceed」を中核とするSaaS企業。売上成長率+24〜28%を維持しながら、PER10.8倍という異常な低評価を放置されている。学校向け有料会員が前年比195%成長（令和8年度）と急拡大しており、BtoS市場への橋頭堡確保は本物だ。しかし「テンバガー候補」として求められる「ビジネスモデルの根本的転換×独占的技術×急拡大する量産案件」の三要件は揃わない。忖度なしに言えば、これは「上場している良い英語学習アプリ会社」であって、構造的破壊者ではない。翌7月14日に通期決算発表が控えており、その内容次第で株価は大きく動く可能性がある。割安成長株としてポートフォリオのごく一部（5%程度）に収める選択はあり得るが、テンバガーのコア銘柄には据えられない。

## 2. 事業概要と競合優位性

- **abceedの実態**：英語学習アプリで1,200冊以上の市販教材にデジタルアクセス可能。AI学習最適化エンジン（弱点分析・問題推薦）搭載。TOEIC受験者の85〜90%が利用という圧倒的な市場浸透。登録ユーザー600万人超（2025年11月時点）、有料会員は約119,000人（FY2026 Q1末）

- **三市場展開**：①個人向け（BtoC/abceed）、②法人英語研修（BtoB）、③中高・大学向け（BtoS/abceed for school）。2025年からアウトバウンドセールスチームを新設してBtoB・BtoS強化

- **競合との差別化実態**：スタディサプリENGLISH（リクルート）はTOEIC特化・大規模広告展開。abceedは教材の幅広さ（1,200冊）と10秒単位の隙間学習対応が強み。ただし本質的な技術差別化は「教材量×AIデータ蓄積」であり、リクルートの資本力で追随可能なリスクあり

- **法人顧客の質**：三井住友信託銀行など大手金融機関の導入実績あり。BtoS市場では全国の中高に展開可能な体制を整備（教科書採択不要化）

- **マクロ政策との絡み**：文部科学省の英語4技能評価政策（スピーキング・ライティング・リーディング・リスニング）が直接追い風。AI添削・発音スコアリング機能が教員の採点負荷軽減に貢献。政府目標（中高生の60%がCEFR A1〜A2以上）達成の実務的ツールとして機能

## 3. 財務分析（直近3期）

- **売上高推移**：FY2024推計≒1,288百万円 → FY2025確定1,650百万円（+28.1%）→ FY2026通期予想2,040百万円（+23.6%）。3期連続20%超成長は安定的な構造成長の証明

- **営業利益推移と課題**：FY2024推計≒319百万円（率≒24.8%）→ FY2025確定400百万円（率24.2%）→ FY2026通期予想330〜440百万円（不確定）。Q1-Q3累計で前年比▲1.1%の微減益。先行投資（アウトバウンドセールス・AI機能開発）により利益成長が売上成長に追いついていない

- **バランスシート・CF**：ROE予想26.55%、ROA予想12.53%は高水準。自己資本比率・有利子負債の詳細は非公開だが、SaaS構造のため重設備投資不要でFCFはプラスと推計。H1経常利益が5%減とあり、投資先行フェーズが明確

- **現在のバリュエーション整理**：時価総額32.74億円（株価645円）。売上2,040百万円に対しPSR約1.6倍はSaaS企業として著しく低評価（通常PSR3〜10倍）。PER10.8倍×成長率24%はPEG比率0.45と理論上は非常に割安

- **見立て**：「伸びる銘柄のブレイク」に近い。ただしFY2026に利益率が低下（先行投資）しており、黒字成長の継続確認が必要。テーマで持ち上げられた面もなく、実力ベースの成長株と判断

## 4. 成長ドライバーの検証

- **学校向け（BtoS）195%成長**：FY2026の最大のポジティブサプライズ。AI添削・発音スコアリング・英会話機能が学校ニーズにマッチ。教科書採択不要化により全国全中高への拡販が可能になった。これが来期（FY2027）の収益貢献として顕在化すれば成長加速の可能性

- **AI機能拡充の継続**：英語作文AI添削、発音スコアリング、英会話AI機能を追加。これらがabceedの付加価値を高め、解約抑止（チャーン低下）と単価向上に貢献するか注目

- **中期経営計画**：詳細な数値目標は未公開情報のため確認できず（IR資料への直接アクセス不可）。ただしFY2026に通期売上2,040百万円（前期比+24%）をコミットしており、成長軌道は明確

- **カタリスト（7月14日通期決算）**：翌日（7/14）に通期決算発表予定。Q3まで売上+24.9%増のペースが維持されており、通期の売上達成は確実な見通し。問題は利益面。上方修正が実現すれば株価回復の起点となる可能性が高い

- **英語アニメコンテンツ展開**：2026年6月からアニメ「ちびまる子ちゃん」の英語吹替版を配信開始（最新ニュース）。コンテンツ充実による差別化とユーザー拡大を狙う動き

## 5. リスク分析

- **最大のダウンサイドシナリオ**：TOEIC・英検などの資格試験の大規模中止・縮小。abceedの収益モデルは試験対策需要に大きく依存しており、COVID-19型の試験中止が起きれば売上が急減する構造的脆弱性。同様に少子化による学習人口減少も長期的リスク

- **競合大手（スタディサプリ/リクルート）の脅威**：リクルートは圧倒的な資本力・営業力を持ち、スタディサプリENGLISHはBtoB展開も積極的。差別化の本質が「教材数×AIデータ蓄積量」にある場合、大手が同等の体制を構築すれば競争優位が消える可能性

- **解約率（チャーンレート）の不透明性**：解約率は非公開。SaaS企業にとって最重要KPIの一つが開示されていない点は投資判断上の大きな不確実性。法人解約率が高ければ、見かけの成長率は「入れ替わり」に過ぎない

- **顧客集中リスク**：主力事業abceedへの高依存（リスクとして自社開示）。ABCEED ENGLISHなどの多角化は始まっているが収益規模は小さく、主力の停滞・縮小が直接業績に響く構造

- **先行投資による利益率低下の長期化リスク**：FY2026にアウトバウンドセールスチームを新設し、AI機能開発投資も継続。成長投資が利益を圧迫する期間が続けば、「成長するが稼がない」会社と市場に認識されPER/PSRが更に低下するリスク

## 6. バリュエーション＆テクニカル

- **現在指標と同業比較**：株価645円、PER10.8倍（予想）、PBR3.24倍、時価総額32.74億円。EdTech・英語学習関連の上場類似企業（EduLab 4427等）と比較してPERは同等〜割安水準。スタディサプリを運営するリクルートHDはコングロマリットのためPER直接比較不可。純粋なSaaS企業との比較ではPSR1.6倍は著しく低く、マーケットが成長に懐疑的か

- **適正株価試算（12〜24ヶ月、FY2027基準）**：
  - **保守シナリオ**（成長15%、利益率20%維持）：EPS69円×PER15倍 = **1,035円**
  - **中立シナリオ**（成長20%、利益率22%）：EPS80円×PER18倍 = **1,440円**
  - **強気シナリオ**（成長30%、BtoS急拡大で利益率改善）：EPS94円×PER22倍 = **2,068円**
  - ※いずれも発行済株式数508万株として計算

- **現在の株価位置**：52週高値980円（2026年4月22日）から▲34.2%下落。52週安値582円（2026年3月24日）からはわずか+10.8%上。高値圏から大きく調整中で、RSI14日が22.5という売られすぎゾーン。25日移動平均乖離率は▲13.7%。2026年6月22日に三角保ち合い上放れを形成したが、その後再び下落した形

- **エントリーゾーン推奨**：580〜650円が適正な打診買いゾーン。52週安値（582円）を強くサポートとして意識しており、この水準を維持する限り下値は限定的と判断。通期決算（7/14）でポジティブな数字が出ればショート勢の巻き戻しが入りやすい水準
