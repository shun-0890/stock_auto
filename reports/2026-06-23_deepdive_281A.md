# 詳細調査レポート：281A インフォメティス（2026-06-23）

## エグゼクティブサマリー

**忖度なしの評価：「テンバガー候補として最有力。ただし2025年の大幅減収リスクを直視せよ」**

NILM技術のIEC国際標準規格（IEC 63297:2025）取得という圧倒的な技術的堀を確立し、東京電力・関西電力・日立・ダイキンといった大企業との資本・事業提携網を持つ。2026年から本格化する次世代スマートメーター展開がNILMのボトルネック（センサー設置工事）を消滅させ、構造的な収益化加速の扉を開く。2026年1Qで経常黒字転換（+260万円）という数字も出た。ここまでは理想的だ。ただし2025年12月期に売上高が前期比▲46%・純損失▲7.2億円という事実は看過できない。業績変動の激しさはリスクとして厳格に管理すること。今日の-3.5%調整局面は、本物のテンバガー候補へのエントリー機会として評価する。

**深掘り調査を経た評価修正：**
- 「独自技術★5」は**さらに確信度が上がった**。IEC国際標準規格化（2025年6月）とNEC特許譲受（2025年5月）により、競合がNILMで追随することが制度的・技術的に著しく困難になった。この評価は揺らがない。
- 「量産案件・受注確度★3」を★3.5に引き上げたい（★4まではいかない）。次世代スマートメーター展開という構造的トリガーが現実化しており、2026年Q1の+72%成長はその序章である。
- 最大の懸念：2025年12月期の**売上▲46%という大幅減収の原因が不明瞭**。詳細な要因（東電との契約一時中断か、開発フェーズの切り替えか）を次の決算説明で必ず確認すること。

## 2. 事業概要と競合優位性

- **NILM技術（Non-Intrusive Load Monitoring）の本質的優位性**：家庭の分電盤主幹を1箇所測定するだけで、AIが電流波形を解析し家電別の使用状況をリアルタイム推定する。追加センサー・工事が不要という「非侵入型」こそがビジネス展開上の最大差別化要因。2025年6月、経産省6年受託の成果としてIEC国際標準（IEC 63297:2025）が世界初発行済み。特許ポートフォリオも日米で堅固。
- **資本・事業提携の厚み**：東京電力PG（合弁会社エナジーゲートウェイ設立、持分40%）、関西電力、日立製作所、ダイキン工業、博報堂DY、伊藤忠エネクス、フォーバル（持分10.95%）、建設技術研究所という上場大企業中心のエコシステムは、後発が短期間で構築できない強固なモートを形成している。
- **競合の実態**：国内に実質的な直接競合は存在しない。海外ではBidgely（米国、B2B型）・Sense（米国、ハードウェア依存型）が存在するが、IEC標準化と日本電力エコシステムへの深い統合という点でインフォメティスは国内での独占的地位を持つ。
- **次世代スマートメーターがゲームチェンジャー**：2026年1月から全国更新が開始。NILM展開の最大ボトルネック（物理センサー設置工事）が構造的に消滅することで、全国8,000万世帯規模の市場への展開が現実的になる。
- **GX政策との直結性**：政府の2050年カーボンニュートラル達成に向けたGX推進投資（10年150兆円超）のインフラ整備として、NILM技術と次世代スマートメーターは直接的な政策的位置付けにある。

## 3. 財務分析

| 期 | 売上高 | 営業損益 | 純損益 | 前期比（売上） |
|---|---|---|---|---|
| 2024年12月期 | 約10.0億円（推定） | 約+7,800万円（推定） | 情報なし | — |
| 2025年12月期 | **5億3,001万円** | 大幅赤字 | **▲7億2,163万円** | **▲46.0%** |
| 2026年12月期（通期予想） | 8億4,500万円 | ▲3億9,500万円 | 情報なし | **+59.4%** |
| 2026年1Q（実績） | 2億300万円 | ▲7,402万円 | — | **+72.0%（前年同期比）** |

- **2025年12月期の大幅減収（▲46%）は最大のリスクシグナル**：黒字が見込まれていたIPO時予想から急転直下の赤字転落。具体的な要因の開示が限定的であり、東電エナジーゲートウェイ絡みの受注スケジュールのズレか、開発フェーズの切り替えの可能性が高い。業績変動が大きいことは認識必須。
- **2026年1Qの経常黒字転換（+260万円）**は信号として重要：前年同期比+72%成長、次世代スマートメーター関連受託案件の増加が主因と示されており、反転上昇の始まりの可能性がある。
- **自己資本比率35.3%**は許容範囲内だが、赤字継続で純資産が侵食されており、資金調達（増資）リスクを常に意識すること。
- **「伸びる銘柄のブレイク」か「テーマで持ち上げられただけ」か**：2025年6月のIEC標準化ニュースでのストップ高（1,435円）は一時的な材料株的反応だった可能性があるが、2026年1Qの業績回復という実績が伴い始めており、本物の構造的成長フェーズへの移行が始まったと判断する。
- FCF・有利子負債の詳細数値は公開情報から確認できず。赤字継続中のためFCFはマイナスと推定。

## 4. 成長ドライバーの検証

- **次世代スマートメーターの本格展開（2026年〜）**：これが最大のカタリスト。全国8,000万世帯へのNILMサービス展開を物理的に可能にする。エナジーゲートウェイ（TEPCO PG×インフォメティス合弁）が展開の主力エンジンとなる。2026年1Qの売上+72%はその初動と解釈できる。
- **IEC国際標準規格（IEC 63297:2025）の活用**：経産省と6年間取り組んだ国際標準化が完了したことで、欧州・グローバル市場での展開が大幅に加速しやすくなった。英国ケンブリッジの欧州拠点（Informetis Europe Ltd.）が実動段階に入る可能性。NEC特許の譲受（日米2件）もグローバル展開の布石。
- **フォーバルとの中小法人向け脱炭素サービス**：2025年12月に商業展開開始、2026年以降全国展開予定。フォーバルは持分10.95%の大株主でもあり、本気度が高い。中小法人市場（BtoBtoCではなくBtoB直接）という新収益軸の立ち上がり。
- **2026年Q2決算（2026年8月予定）が次の重要マイルストーン**：通期経常損失計画▲3.5億円に対して1Qで経常黒字+260万円を達成しており、通期計画の大幅上振れが視野に入る。Q2での業績トレンドが確認できれば、市場の評価が大きく変わるトリガーになる。
- **経営陣のコミットメント**：ソニー由来のNILM研究を2013年に独立させ、IEC標準化まで12年間取り組み続けた只野太郎代表の長期コミットメントは、技術の本物度を示す指標として評価できる。ただし中期経営計画の具体的数値目標（3〜5年）の開示は確認できず、投資家向けの定量的なコミットが弱い点は改善を求めたい。

## 5. リスク分析

- **最大のダウンサイドシナリオ**：次世代スマートメーター展開が遅延・縮小し、エナジーゲートウェイ経由の受注が再び停滞。2025年12月期のような大幅減収が繰り返されると赤字が累積し、増資（希薄化）が避けられなくなる。株価は大幅下落するシナリオ。
- **業績変動リスク**：2024年期（黒字圏推定）→2025年期（▲46%減収・大赤字）という急落は、単一の大型顧客・プロジェクトへの依存度が高い構造を示唆する。BtoBモデルゆえの受注集中リスクは継続的に監視が必要。
- **競合の参入リスク**：グローバルではBidgely・Senseが存在するが国内への本格参入は限定的。ただしNTT・ソフトバンク等の国内大手がエネルギーDX分野で独自のAI分析技術を展開するリスクは実在する。IEC標準が参入障壁として機能するかが鍵。
- **マクロリスク（本日の環境）**：日銀利上げ（1.0%）によるグロース株バリュエーション圧縮、イラン情勢による原油再騰（エネルギーコスト上昇は電力会社の投資余力を削ぐ可能性）。特にエナジーゲートウェイ（東電合弁）の投資ペースが政策・原油環境の変化で鈍化するリスク。
- **流動性リスク**：発行済株式数約580万株・時価総額43〜46億円の小型株は、機関投資家が大量ポジションを取りにくく、個人投資家の需給で株価が乱高下しやすい。2025年6月のストップ高（+47%）後の急落はその典型。

## 6. バリュエーション＆テクニカル

| 指標 | 数値 |
|---|---|
| 株価（参考：2026年5月15日） | 約765円 |
| 時価総額 | 約43〜46億円 |
| PBR | 5.05倍 |
| PER | 算出不可（赤字継続） |
| EV/EBITDA | 算出不可（EBITDA赤字） |
| 52週高値 | 1,435円（2025年6月30日・IEC標準化ニュースでストップ高） |
| 52週安値（推定） | 約600〜700円帯 |

- **適正株価の試算**（PSRベース。赤字のためPER不可、2027年以降の売上推移を見据えて試算）：
  - 保守シナリオ（2027年売上15億円・PSR 3倍）：時価総額45億円 → **株価 約776円（現状維持水準）**
  - 中立シナリオ（2027年売上20億円・PSR 4倍）：時価総額80億円 → **株価 約1,380円**
  - 強気シナリオ（2028年売上40億円・PSR 5倍。次世代スマートメーター全国展開加速）：時価総額200億円 → **株価 約3,450円**
- **テンバガーシナリオ（36〜48ヶ月）**：次世代スマートメーター全国8,000万世帯へのNILMサービス展開が本格化し、ARRが急増して100億円規模の売上を達成（PSR 3〜4倍）→ 時価総額300〜400億円 → **株価 5,000〜7,000円**（現在比6〜9倍）。これは達成可能なシナリオだが、2026〜2027年の業績が順調に回復することが前提条件。
- **テクニカル**：52週高値1,435円から約▲47%下落した水準（765円前後）にある。IEC標準化バブルが剥げた後の適正な評価水準に戻ってきた段階と解釈できる。本日の日経▲3.5%の調整で、さらに下値が広がっていれば買い増し好機。移動平均線の状況は日次データ取得不可のため要確認。
- **同業比較**：ENECHANGE（4169・電力切り替えプラットフォーム）が国内最近傍だが、NILMという技術軸は独自領域のため直接比較は困難。グローバルのNILM企業は未上場のため参照可能なバリュエーション指標なし。

## 投資判断

- **推奨：条件付き買い（テンバガー候補として積極評価）**

- **エントリータイミング：** 本日（2026年6月23日）の日経▲3.5%調整局面をエントリーに活用。現在株価水準（765円前後、ただし本日さらに下落している可能性あり）で予算の1/3を今日の引け〜翌営業日寄りで買い。その後、**700円以下まで下落した場合に追加1/3**、**2026年Q2決算（2026年8月）での業績回復確認後に残り1/3**を積み増し。マクロはイラン情勢リスクがあるため、一括投入は避けること。

- **推奨ポジション：** ポートフォリオの7〜10%（テンバガー候補の中核に据えるに値する。ただし業績変動リスクを踏まえて10%は上限とする）。

- **目標株価（12〜24ヶ月）：** 1,200円（保守）/ 1,800円（中立）/ 3,500円（強気）

- **損切りライン：** 550円（52週安値推定600〜700円帯を大幅に下回った場合。次世代スマートメーター関連の受注悪化や増資リリースが出た場合は即時撤退を検討）

- **注視トリガー（時系列順）：**
  - **2026年7月〜8月：2026年Q2決算発表**（最重要）→ 経常黒字継続かつ売上が通期計画の50%以上（4.2億円）に達していれば強気シフト
  - **2026年通年：次世代スマートメーター導入台数・エナジーゲートウェイの受注状況**（東電の設備投資計画と連動）

  - **2026年下半期：IEC標準化を活用した欧州展開の具体的な契約発表**（Informetis Europe Ltdの動き）
