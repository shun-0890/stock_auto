# 詳細調査レポート：4258 網屋（2026-07-03）

## エグゼクティブサマリー

**忖度なしの評価：「現環境下で最も筋の通った構造的成長銘柄。段階的積み上げを推奨する」**

網屋は、独自特許を持つログ管理SaaS「ALog」とネットワークセキュリティ「Verona（Network All Cloud）」という二軸のサブスクリプション事業を中核とし、4期連続で19〜34%の売上成長、2025年は営業利益を前年比+99.8%と倍増させた実績がある。金利上昇・グロース株逆風という現マクロ環境でも、サイバーセキュリティは「必須インフラ」として予算削減されにくく、景気耐性が高い。2026年7月にはAI搭載の「ALog V2」をリリース、5月にはAMPEREを子会社化して「インダストリアルセキュリティ事業」という第3の成長柱を追加した。PER34.7倍（2026年6月時点）はグロース市場全体の売り圧力を考慮しても割安感があり、現在が段階的なポジション構築の好機と判断する。ただし、FY2026の会社予想が成長率鈍化（売上+18%、営業利益+14%）を示している点は精査が必要で、盲目的な強気は禁物だ。

## 2. 事業概要と競合優位性

- **主力製品①ALogシリーズ（データセキュリティ、売上構成42%）**：ログ管理SaaS。特許技術「自動ログ変換」により、複雑なシステムログを「誰が・何を・いつ」という人間が即理解できる形式に自動変換。2026年7月に「ALog V2」リリース——AI異常検知・高度可視化機能追加でアップセル機会が大幅拡大
- **主力製品②Network All Cloud（ネットワークセキュリティ、売上構成58%）**：フルマネージドSASE「Verona」＋クラウド無線LAN「Hypersonix」。AMPEREの子会社化でVeronaのルーター調達を内製化→コスト構造改善＋品質管理強化
- **競合優位性**：グローバルのSIEM大手（Splunkは約4,000億円規模でCiscoが4兆円で買収）と異なり、日本中小企業が導入できる「適正価格×運用負荷ゼロ」のポジショニング。大手SIEMが対応しない90%の日本企業をターゲット市場として取り込む
- **顧客・取引先**：非開示の部分が多いが、中小〜中堅企業を中心に幅広く分散。金融機関・製造業・流通など規制対応が求められる業種での採用が進む
- **マクロとの絡み**：政府の「経済安全保障」・「重要インフラのサイバー対策強化」方針と完全整合。経済産業省のセキュリティ強化推進ガイドラインがALog採用を後押し

## 3. 財務分析（直近4期＋予想）

- **売上高推移（億円）**：FY2022 *29.9* → FY2023 *35.6*（+19.2%）→ FY2024 *47.7*（+33.9%）→ FY2025 *59.4*（+24.5%）→ FY2026予 *70.0*（+18.0%）。4期連続の安定した二桁成長が持続
- **営業利益推移（億円）**：FY2024 *約5.3* → FY2025 *10.51*（+99.8%、利益率17.7%）→ FY2026予 *12.00*（+14.1%）。2025年の利益倍増は「全サブスク化完了」による構造的コスト改善が主因
- **Q1 FY2026（2026年1〜3月）**：経常利益331百万円（3.31億円）。四半期ベースの年換算で13.2億円と、通期予想12.00億円を上回るペース。**会社予想は保守的の可能性が高い**
- **ROE・自己資本比率**：ROE 27.14%（2025実績）は中小型SaaSとして高水準。2023年末時点で自己資本比率47.1%（総資産37.8億円、自己資本17.8億円）と財務健全性も良好
- **「伸びる銘柄のブレイク」か**：**YES**。4期連続成長＋2025年に利益が急加速した局面は典型的なSaaSの「利益ブレイクアウト」。サブスク転換の完了が利益率構造を変えており、一過性でない

## 4. 成長ドライバーの検証

- **ALog V2（2026年7月リリース）**：AI異常検知・高度可視化という機能追加がアップセルを促進。既存顧客のARPU（平均単価）上昇→売上に反映されるのはH2 2026〜2027。成長再加速の最重要トリガー
- **AMPEREの子会社化（2026年5月）**：産業機器メーカー子会社化で「インダストリアルセキュリティ」参入。取得額5.4億円（純資産5.4億円のほぼ等価取得）は低リスクM&A。Veronaのルーター調達内製化でコスト削減かつサプライチェーンリスク低減
- **中期経営計画（2025年3月開示）**：最終年度FY2025にARR目標42億円以上を設定（全サブスク化を前提）。2025年実績59.4億円の売上うち大半がARR化すれば目標を超過達成した可能性が高い
- **サイバーセキュリティ需要の構造的加速**：経済安全保障・AI普及による攻撃高度化・経産省ガイドライン強化が重なり、2026〜2028年にかけて国内需要はさらに加速する見通し
- **Q1 2026のデータセキュリティ事業+30%超**：ALog主力のデータセキュリティが高成長を維持。これがALog V2のリリース前の数字である点が重要——リリース後の上積みが期待できる

## 5. リスク分析

- **最大のダウンサイド**：FY2026会社予想（売上+18%、営業利益+14%）が示す「成長鈍化」の現実化。万一、AMPERE統合コストや新規投資で利益が会社予想を下回れば、PER35倍の前提が崩れ株価急落リスクあり
- **競合の参入・代替リスク**：Palo Alto Networks、CrowdStrikeなどグローバル大手が日本市場の中小企業向けに価格を引き下げてくる可能性。グローバルベンダーのローカライズ投資が進めば、「日本中小特化」の優位性が薄れる
- **AMPERE統合リスク**：インダストリアルセキュリティは網屋のコア事業と異なる。産業機器会社の経営・人材・文化の統合に失敗すると、収益への悪影響と経営資源の分散が発生
- **顧客集中リスク**：非開示のため確認困難。ただし売上70億規模で特定顧客への高依存は構造上考えにくく、分散型と推定
- **マクロリスク**：金利上昇（グロース株PER圧縮）は継続リスク。ただしセキュリティ需要は景気耐性が高く、同業他社比較でも下落率は限定的と見る。日銀の追加利上げ（9〜12月）でグロース市場全体が下落する局面は買い増し機会と捉える
