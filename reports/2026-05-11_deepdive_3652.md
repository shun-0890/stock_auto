# 詳細調査レポート：3652 ディジタルメディアプロフェッショナル（DMP）（2026-05-11）

## エグゼクティブサマリー

**忖度なしに言えば「強気だが確認事項が残る。ハイリスク・ハイリターン」と評価する。**

DMP（ディジタルメディアプロフェッショナル）は日本唯一の独立系GPU IPライセンサーであり、エッジAI半導体領域でのIPライセンス事業という観点では、中長期的にARMモデル（CPUコアのIPライセンス）に近い超高収益・高倍率評価が期待できるビジネスを持つ。SOX指数+5.5%急騰の直接的恩恵を受ける数少ない国内小型銘柄として、時価総額116億円でのテンバガー余地は最大クラス。ただし、FY2026 Q2に営業損失292Mを計上した事実と、新しい主力製品（台湾iCatch Technology社との共同エッジAI半導体）の商業的実績が未検証である点が重大な不確実性。「理論的最高評価」と「実績の希薄さ」が同居する特殊銘柄。

---

## 1. 構造的成長ストーリー判定

### 判定：**B（短所あり）**

| チェック項目 | 評価 |
|------------|------|
| 独自技術（他社依存リスク） | ◎ SMAPH GPU IPは独自特許。ただしiCatch案件では台湾企業依存 |
| ビジネスモデルの根本的進化 | ◎ ARMに近い高利益率IPライセンスへのシフト方向性は正しい |
| 成長段階にある市場 | ◎ エッジAI半導体は超構造的成長（SOX+5.5%が証明） |
| 安定供給の見込み | △ 主力製品がアミューズメント向けLSI（縮小業界）からエッジAIへ移行中で過渡期 |
| 新規量産案件 | △ iCatch共同エッジAI半導体はFY2026 Q4から販売開始も商業実績ゼロ |
| 利益率成長 | ▲ FY2026 Q2に営業損失292M。薄利（0.6%）での着地見通しは脆弱 |

**評価コメント**：ビジネスモデルの方向性（エッジAI IP → ARMモデル）は理想的だが、現時点で「構造的成長の証拠」となる確定収益がない。QPSHDが「697億円の確定契約」という動かぬ証拠を持つのに対し、DMPはまだ「可能性の段階」。ただしこの「可能性」の規模が大きく、エッジAI産業のTAMを考えれば成功時のリターンは桁違い。

## 2. 事業概要と競合優位性

### 主力製品・サービス

**① GPU IP コアライセンス事業（主力・高付加価値）**
- SMAPH-F / SMAPH-S / SMAPH-H：車載・産業機器向け独自GPU IPコア
- ZIA MOVE：自律移動ロボット（AMR）向けAI制御IPライセンス
- 過去の実績：PICA200がNintendo DS / 3DSに採用（ゲーム半導体IP市場での先行者）

**② LSI製品事業（縮小傾向）**
- アミューズメント機器向けの独自GPU組み込みLSI製造・販売
- 業界縮小に伴い売上比率は低下中。主力からの移行が課題

**③ 次世代エッジAI半導体（新事業・最重要カタリスト）**
- iCatch Technology（台湾）との戦略的パートナーシップ（2024年10月締結）
- 両社の技術を融合したエッジAI半導体を開発
- FY2026 Q4（2026年1〜3月）から販売開始済み
- ターゲット市場：産業機器、車載、スマートホーム、ロボット

**④ AIソリューション事業**
- 製造業向け画像検査AI、建設機械向けモデルベース開発ツール（RTMaps）
- プロフェッショナルサービス収益

### 競合優位性と懸念点

| 要素 | 評価 | コメント |
|------|------|---------|
| GPU IP技術 | ◎ | 国内唯一の独立系GPU IPライセンサー |
| iCatch提携 | ○ | 台湾の実績あるSoC設計会社との協業。開発リスク分散 |
| Nintendo実績 | ○ | PICA200の採用実績は信頼性証明だが過去の話 |
| アミューズメント依存 | ▲ | 縮小市場への依存は長期的なリスク |
| iCatch依存リスク | △ | 次世代製品が台湾企業依存。独自開発100%ではない |

### マクロとの絡み
SOX（半導体指数）が+5.5%急騰した本週は、エッジAI半導体への市場期待を如実に示す。DMPが提供するGPU IPはエッジAI処理の根幹技術であり、この相場環境で再評価が起きやすい。

|----|--------|---------|-----------|
| FY2023（3月期） | 25〜28億円程度 | 黒字 | — |

> **重大な注意点**：FY2026 H1（4-9月）に292M の営業損失を計上。H2（10-3月）にiCatch製品立ち上げ + 回復で年間20M黒字に持ち込む計画だが、振れ幅が大きい。

### 財務的特徴
- **ファブレスモデル**：製造設備なし、R&D投資が主体のため高利益率ポテンシャルを持つ
- **IPライセンス比率向上**：LSI事業縮小とIP比率上昇は利益率改善に寄与する方向
- **自己資本比率**：十分に健全（製造業でないため固定資産少ない）
- **時価総額 vs 売上**：116億円 / 32.5億円 = PSR 3.6x → ファブレス半導体IPとして「適正〜やや高い」水準

### 「伸びる銘柄のブレイク」か「テーマで持ち上げられただけ」か
→ **判定保留（グレーゾーン）**。エッジAI半導体という本物のテーマに乗っているが、収益化の実証が不十分。FY2027 Q1-Q2（2026年4-9月）の決算がこの問いに答える。

## 4. 成長ドライバーの検証

### 最重要カタリスト：iCatch共同エッジAI半導体の商業展開

**背景**：
- iCatch Technologyは台湾の実績あるSoC設計メーカー（IP系）
- DMP独自のGPU IP + iCatchのSoC設計力を組み合わせた次世代エッジAI半導体
- FY2026 Q4（2026年1〜3月）に販売開始→ FY2027（2026年4月〜）が最初の本格展開期

**強気シナリオ**：
- FY2027でエッジAI製品が年間10〜15億円の新規収益
- 既存IP事業 + AI製品で総売上45〜50億円
- 利益率が10%+に改善→ 営業利益4.5〜5億円
- 時価総額が現在の116億円から300億円+ へ（EV/EBITDA 30x）

**弱気シナリオ**：
- エッジAI製品の採用が遅れ、FY2027も薄利
- アミューズメント縮小が続き売上伸び悩み
- 株価は2,500〜3,000円レンジへ

### 次世代エッジAI市場の大きさ
- 2025年のエッジAI半導体市場規模：約180億ドル（~2.8兆円）
- 2030年：約690億ドル（~10.8兆円）まで成長予測
- DMPが仮にこの市場の0.1%を取るだけで108億円規模の市場

### 直近のIR・ニュース
- 2024年10月：iCatch Technology との戦略的パートナーシップ発表
- 2026年3月期Q4（1〜3月）：エッジAI半導体の販売開始
- FY2027の本格収益化が最大の注目点

## 5. リスク分析

### ★★★ 最大リスク：iCatch製品の商業展開失敗
- 主力新製品が顧客に採用されなければ収益化不可
- 半導体製品の採用決定は長いリードタイム（6〜18ヶ月）を要する
- 採用されたとしても量産立ち上げまでに数期かかる

### ★★★ リスク：収益の構造的脆弱性
- FY2026 Q2に-292M の損失は「まだ稼げていない」現実を示す
- 年間売上32.5億円で利益率0.6%は小さな問題で赤字転落しうる

### ★★ リスク：アミューズメント市場の更なる縮小
- 国内アミューズメント向けLSI（パチスロ等）は規制強化・市場縮小が継続
- この依存度が高いほど成長の足を引っ張る

### ★★ リスク：競合からの挟み撃ち
- エッジAI IPは台湾・韓国・米国のファブレス企業が積極参入
- ARM、NVIDIA（エッジ向け）との競争で独自IP価値が希薄化するリスク

### ★ リスク：台湾（iCatch）への供給依存
- iCatchとの関係が破綻した場合、次世代製品の基盤が失われる

## 詳細分析要点

・DMPは「日本唯一の独立系GPU IPライセンサー」という圧倒的な希少価値を持ち、エッジAI市場の成長はその価値を指数関数的に高める可能性がある。
・FY2026 Q4から販売開始のiCatch共同エッジAI半導体が本命。FY2027が最初の本格評価年度。
・最大の懸念はFY2026 Q2での-292M営業損失。薄利での着地見通しは財務的な脆弱性を示す。
・52週安値（1,940円）からすでに+91%の上昇。スクリーニング条件「+40%超は除外」に抵触する点で注意。現在値からの新規買いは「高値追い」の側面があり押し目待ちが原則。
・アミューズメント向けLSI（縮小事業）の比率を早期に下げることができるかが経営の鍵。
・SOX+5.5%急騰の相場環境はDMPの株価追い風だが、本質的には「AIエッジ製品の採用実績」が株価の正当性を担保できるかどうかが問われる。
・QPSホールディングス（464A）と比較すると、QPSHDは「697億円案件確定」という確実性があるのに対し、DMPは「可能性はあるが未証明」という段階の差がある。
・投資判断としては、打診買いを入れつつFY2026通期決算（5月発表予定）でiCatch製品の実績を確認、良好なら比率を引き上げるアプローチが適切。
・テンバガーポテンシャルは理論上最大クラス（強気シナリオで15,000円）。ただしリスクも最大クラス。
