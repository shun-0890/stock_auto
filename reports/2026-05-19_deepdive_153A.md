# 詳細調査レポート：153A カウリス（2026-05-19）

## エグゼクティブサマリー

忖度なしの「最強気」である。

国内唯一の金融機関横断型不正検知データベースというネットワーク効果型モートは、資本力でも模倣できない真の競争優位。FATF第5次相互審査（2028年オンサイト）を前にした金融庁義務化は不可逆的需要を創出し続けており、規制という「永続的な追い風」を背景にした収益成長は構造的だ。

ARR+24.3%・チャーン0.28%・粗利率63%という「SaaS高品質指標三冠」は国内SaaS銘柄でもトップクラス。2025年12月期の先行投資フェーズを経て2026年Q1で増益転換（+33.9%）したことは、成長再加速の初動であり最も理想的な仕込みタイミングである。

加えて、Grid Data KYC（電力契約情報を活用した本人確認）という第2の収益柱候補が業績予想の外に潜在する。これが本格受注段階に入ればARR成長率は30〜40%台への加速も現実的であり、現在のバリュエーションには「新事業オプション価値がほぼゼロ」で折り込まれている。時価総額130億円超へ上昇したため「発見の最早期」は過ぎているが、国内銀行浸透率15%→50%という残存TAMの大きさと複数の上振れ材料を考えると、テンバガー候補の中核として今が最良の積み上げ機会と確信する。

## 2. 事業概要と競合優位性

- **主力はFraud Alert（不正検知SaaS）**：口座開設・ログイン・入出金の3タッチポイントを250超の独自パラメータ（端末指紋・IP・行動パターン等）でリアルタイムスコアリング。月間6億件処理、不正検知率約1.77%（月間1,060万件相当）。2024年12月に「入出金検知」機能を正式リリースし資金移動監視に拡充。
- **決定的差別化は「業界横断型ブラックリストDB」**：NTTデータ・NEC・SAS・TRUSTDOCKなどすべての競合は単一機関向けの閉じたシステムであり、業界横断DBを保有しない。カウリスだけが複数業態（銀行・証券・クレカ・暗号資産）の不正者情報を統合しており、参加機関が増えるほど検知精度が指数的に向上する正のフィードバックループを持つ。
- **顧客基盤の質**：千葉銀行・東京スター銀行・百五銀行・SBIレミットなど金融機関40社超。国内銀行採用率約15%、目標2026年末50%（残存TAM大）。上位10社で売上の約66.2%を占める集中リスクあり。
- **FATF義務化という不可逆的追い風**：2019年のFATF第4次対日審査でマネロン対策不備を指摘され「優先フォローアップ国」に指定。金融庁は全金融機関にリスクベースAML態勢整備を義務付け。2028年FATF第5次オンサイト審査を前に義務化対応ニーズは2026〜2027年が最大化フェーズ。Grid Data KYCは規制サンドボックス制度を活用した国内初の事業化で、KYC強化義務化ともリンクする。

## 4. 成長ドライバーの検証

- **Fraud Alert深耕（主幹エンジン）**：国内銀行採用率15%→50%という残存TAMで、既存顧客アップセル（プラン拡張・監視トランザクション数増加）を組み合わせたランドアンドエクスパンドモデルが機能中。Q1の増収+22%はほぼこれが牽引。
- **Grid Data KYCという第2の収益柱候補**：電力契約情報8,600万世帯分を活用したKYCサービスは業績予想に「未反映」。2026年3月に飯田信用金庫で実証実験開始。本格受注が始まれば上振れ幅は大きく、ARR成長率が30〜40%台に加速する可能性がある。
- **経営陣のコミットメント（2025年3月の成長可能性開示）**：①Fraud Alert：国内銀行50%浸透、②金融機関以外（通信・電力・EC）への展開、③Grid Data KYCの第2収益柱化、④地方拠点強化（福岡拠点開設済み）という4本柱を明示。
- **配当・自社株買い**：2025年12月期より初配当実施（EPS43.27円）。2026年3月に自社株買いを決議。株主還元姿勢の強化は経営陣の業績自信の表れ。
- **直近の重要ニュース**：2026年5月15日Q1決算（売上+22%・利益+34%）が市場コンセンサスを上回り当日+8.92%急騰。2026年3月の飯田信用金庫Grid Data KYC実証開始が次フェーズへの道標。

## 5. リスク分析

- **最大ダウンサイド：FATF規制の骨抜きまたはメガバンクの内製化**。金融庁義務化スケジュールが緩和された場合や、三菱UFJ・三井住友等が巨大なシステム投資で類似機能を内製化した場合、導入ペースが大幅鈍化するリスク。ただしネットワーク効果（データ資産）は内製では再現不能なため、完全代替は困難と判断。
- **競合参入リスク**：NTTデータ・NEC・富士通の本格参入、またはFeedzai・BioCatch等の外資ローカライズ強化。業界横断DBは短期模倣困難だが、大型資本による長期競合は排除できない。生成AI/LLMによる不正検知精度の汎用化も長期的リスク。
- **顧客集中リスク（上位10社で売上66.2%）**：チャーン0.28%は極めて低く実際の解約リスクは限定的だが、大手金融機関1〜2社の条件変更・契約更新延期が業績への短期影響が大きい。顧客分散の進展を四半期ごとにモニタリングすることが肝要。
- **マクロリスク**：JGB金利2.80%超は金融機関収益拡大要因（プラス面）だが、景気後退局面での設備投資抑制は解約圧力に転じるリスク。為替リスクは国内特化のため現時点で限定的。スタグフレーション長期化は中小金融機関の体力悪化につながるリスクあり。
- **人材獲得競争**：先行投資フェーズに対応できる金融ドメイン×エンジニアの採用難が固定費増加要因として利益率を圧迫するリスク。
